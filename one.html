1. Install Apache on your machine

  2. Create two virtual host ( Name Based )

    a. one with name => one.<yourloginid>.com

    b. other with name => two.<yourloginid>.com

    => For name resolution use /etc/hosts file.

        => create index.html for both vhosts and put the content one.<yourloginid>.com for 1st one and two.<yourloginid>.com for 2nd.

  3. on Virtual host “two”, create a directory named “protected” and deny access to this.

  4. Disable directory list on 1st virtual host ( one )

    => Directory listing should be ON on 2nd virtual host

  5. Hide apache version on all ( Check this by running curl -I and see if apache version is there )
